Displays additional player information, stored in a Google Sheet, inside ESPN Fantasy Baseball Leagues. Each row in the Google Sheet represents a player. Required columns are player first name and player last name. Specify any additional columns to display below.

Google Sheet Shareable Link
Click Share button in Google Sheet to open the Share with others dialog box. Publish the sheet (File->Publish). Then click Get Shareable Link button. Copy the full link and paste it here. Google Sheet Sharing permissions must be set to Anyone with link can view.


Worksheet Number
Uses a single worksheet. Left most worksheet is 1.
